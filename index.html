Introdução

No linux usamos o terminal pra instalar alguns aplicativos , para acessarmos algumas páginas e não só.

  • .deb-extensão usada apenas no linux

Instalação do git

instalamos o git no terminal, acessamos o site do git e baixamos a versão pro linux(alguns aplicativos no linus não instala-se direito) copiamos o código do site,
e colamos no terminal deu erro porque não executamos como administrador(para executarmos como administrador antes do código temos que usar o "sudo") depois copiarmos
o segundo código e executamos,demos continuidade e verificamos a instalação(git -v).

Instalação do Vs code

Depois de irmos ao site e baixarmos o pacote de Instalação, demos continuidade na instalação no terminal.Para instalar usamos o "cd" e a inicial da pasta em que
se encontrava o pacote de instalação, com a pasta acessada usamos o código "dpkg -i"(código de instalação para os pacotes .deb)

Instalação do Mysql

copiamos o código de instalação(a partir do site) para executarmos no terminal usamos o "sudo mysql" depois de executado usamos o "show databases para nos mostrar as as bases de dados existentes" por fim criamos uma base de dados usando o comando create databases.
